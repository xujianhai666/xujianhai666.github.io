Golang For

小伙伴写for循环, 提出了一个问题: 为啥下面的不可以这么写 (虽然是新手问题, 但是还是得从源代码解释):

arr := []int32{1, 2, 3, 4, 5}
for _, a := range arr {
   go func(){
      fmt.Println("a: %v", a)
   }()
}

go build -gcflags '-l' -o main main.go
go tool objdump -s "main\.test0"  main
